O que é o html

(Voltar)

O html ele é muito utilizado para criar pagisnas na web por meio de tags atributos, que definem como o conteúdo deve ser apresentado em um navegador web.

Tags

(Voltar)

as Tags Htmls são utilizadas para formata e marca os conteudos visual do texto em decumentos html cada tag tem um nome e uma sintaxe que dertemina como ela deve ser aberta e fechada

Atributos das tags

(Voltar)

Listas Ordenadas e Não Ordenadas

(Voltar)

as os tipos de listas Ordenadas e não ordenadas são muito utilizadas na hora de fazer um indice, e a diferença entra elas é que a ordenadas sempre vai ter uma numeração e 1 a 10 ou abc, enquanto a não ordenada vai ser sempre marcada por pontos, quadrados e etc.
